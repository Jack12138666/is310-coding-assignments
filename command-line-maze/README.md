## Instructions
1. Start at the `Entrance` directory
2. Read the `Start.txt` before started


## Essential Commands
- `ls` - see what's in the current location
- `cd [directory_name]` - go to that file
- `cd ..` - go back to the previous location
- `cat [filename]`  read clues
- `pwd` - check where you are



## Goal
Navigate the maze using PowerShell commands and find the final solution.
